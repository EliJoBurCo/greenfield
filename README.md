### Installing Dependencies

From within the root directory:

```sh
npm install
```

### Run Locally

1. Run the back-end:

  ```sh
  npm start
  ```

2. Build the front-end (in another tab):

  ```sh
  npm run dev
  ```

3. Visit `http://localhost:8080` (or whichever port the server started on).
